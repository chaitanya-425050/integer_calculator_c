# Integer Calculator in C

A command-line integer calculator written in C.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus operation
* Integer input validation
* Operator validation
* Division by zero protection
* Continue or quit functionality

## Technologies Used

* C Programming Language
* Standard C Library

## How to Compile

```bash
gcc integer_calculator.c -o calculator
```

## How to Run

```bash
./calculator
```

## Sample Operations

* 10 + 5 = 15
* 10 - 5 = 5
* 10 * 5 = 50
* 10 / 5 = 2
* 10 % 3 = 1
